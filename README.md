## Wall



A modern Django REST Framework project with full-featured JWT authentication using phone numbers and OTP verification. This backend supports CRUD operations for ads and user accounts, leverages Redis for OTP caching, includes permission management, and is fully Dockerized for easy deployment.



--- Features



- JWT Authentication via phone number



- OTP Verification** cached in Redis



- CRUD Operations** for ads and users



- User Permissions** applied across the project



- Dockerize for consistent local and production environments



- Dynamic Pagination and Search



- Query-based Search** on ads
